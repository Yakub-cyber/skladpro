# СкладПро — AI Warehouse

Веб-система складского и оптового учёта с ИИ-функциями. Аналог CloudShop, доработанный под оптовый склад. Работает в двух режимах: **облако** (мультитенант на Supabase — авторизация, компании, роли, синхронизация между устройствами) и **офлайн** (данные в `localStorage`, если ключи Supabase не заданы). Дальше — PWA и мобильное приложение на React Native.

Дорожная карта и статус фич — в [ROADMAP.md](ROADMAP.md).

🔗 **Демо:** https://yakub-cyber.github.io/skladpro/

## Возможности

**ИИ-функции**
- 🧾 **Накладные из текста** — пишешь «Гвозди 100шт, Молоток 5шт» → готовая накладная с ценами для печати
- 🗺️ **Карта склада** — поиск товара подсвечивает ячейку
- 🚚 **Маршрут доставки** — оптимальный объезд заказов по городу (TSP + 2-opt)
- 📦 **Маршрут сборщика** — короткий обход ячеек при комплектации
- 📊 **ИИ-аналитика** — прогноз исчерпания запасов, неликвид, авто-заявки на закупку

**Учёт и операции**
- Товары, остатки, категории, штрихкоды Code128, весовой товар, маркировка «Честный знак»
- Заказы с воронкой статусов и публичным трекингом для клиента
- Складские операции: приёмка со сканером (USB/камера), списание, возврат, инвентаризация
- Печать этикеток и ценников, импорт прайса из Excel/CSV
- CRM с программой лояльности, поставщики, онлайн-витрина с корзиной

**Управление**
- Авторизация по PIN, роли сотрудников (RBAC)
- Кассовые смены с Z-отчётом
- Журнал действий по всем разделам
- Тёмная/светлая тема, командная палитра (Ctrl+K)

## Стек

React 18 · Vite · Tailwind CSS v4 · Zustand · React Router · Recharts · lucide-react

## Запуск

```bash
npm install
npm run dev      # http://localhost:5180
npm test         # юнит-тесты (Vitest)
npm run lint     # ESLint
npm run build    # сборка в dist/
```

Для облачного режима скопируйте `.env.example` → `.env` и подставьте ключи Supabase.
Без них приложение работает офлайн на `localStorage`.
